# 将下列代码单独拷贝到main.c（全部替换）中进行独立功能测试

## 1. test_ENCODER.c - 编码器测试

### 测试步骤：
1. 连接AS5600编码器A到I2C总线1（引脚37-SDA, 36-SCL）
2. 上传代码到开发板
3. 打开串口监视器(115200波特率)
4. 手动旋转编码器连接的电机轴
5. 观察角度和速度输出
6. 注释掉编码器A部分，取消注释编码器B部分
7. 连接AS5600编码器B到I2C总线0（引脚8-SDA, 9-SCL）
8. 重复测试步骤

### 预期结果：
- 串口输出连续的圈数和速度值
- 旋转方向应与输出变化方向一致
- 静止时速度值应接近0
- 两个编码器应独立工作，互不干扰

- 温度数据应在合理范围内
- 数据输出稳定无跳变

## 4. test_FOC_A.c - 单电机FOC控制测试

### 测试步骤：
1. 连接电机A、驱动器和AS5600编码器A
2. 连接电流检测电路
3. 上传代码到开发板
4. 打开串口监视器(115200波特率)
5. 等待初始化完成("Motor ready")
6. 通过串口发送速度命令(如"M10"设置10rad/s)
7. 观察电机响应和监控数据
8. 测试不同速度值
9. 测试急停("M0")

### 预期结果：
- 电机应平稳启动并达到目标速度
- 速度响应应符合PID参数设定
- 监控数据应显示电流、电压等参数
- 电机应能快速响应速度变化
- 无异常噪音或抖动

## 5. test_FOC_AB.c - 双电机FOC控制测试

### 测试步骤：
1. 连接两个电机、驱动器和编码器
2. 上传代码到开发板
3. 打开串口监视器(115200波特率)
4. 等待初始化完成("Motor ready")
5. 通过串口分别控制两个电机：
   - "A10"设置电机A速度为10rad/s
   - "B-5"设置电机B速度为-5rad/s
6. 观察两个电机的同步运行
7. 测试不同速度组合

### 预期结果：
- 两个电机应能独立控制
- 速度响应应符合各自PID参数设定
- 监控数据应区分显示两个电机的状态
- 无交叉干扰现象
- 系统稳定性良好
